📖 What is Nostrcoin?
Nostrcoin (NSTC) is an experimental, educational cryptocurrency built entirely on the Nostr protocol. Instead of a traditional blockchain, every mined block and every transfer is a public Nostr event — anyone can independently validate the full history.

It uses proof-of-work mining: every 10-minute epoch, the first valid event with the right hash difficulty wins a block reward of 50 NSTC. Rewards halve every 210,000 blocks, similar to Bitcoin, with a max supply of 21,000,000 NSTC.

This project has zero monetary value and is meant purely to explore what's possible when you build a currency-like system on top of a decentralized social protocol.

Step 3 of 3
⛏️ Start Mining NSTC
Mining is done right in your browser. Here's how:
  1. Open the Wallet at saccoci.github.io/wallet
  2. Connect your Nostr extension — tap "Connect Extension" and approve the request
  3. Scroll down to the Mining section — you'll see a Start Mining button
  4. Optionally set a message — this gets embedded in your mined block (like a coinbase message in Bitcoin)
  5. Tap Start Mining — your browser will start hashing. Each 10-minute epoch, only one miner wins the block reward of 50 NSTC
  6. When a block is found — it's broadcast to Nostr relays automatically. Your balance updates at the next epoch (~10 min)
💡 Tips

Keep the wallet tab open and active while mining — browsers throttle background tabs. The miner stops automatically at epoch boundaries and restarts fresh each epoch. Check the Explorer to see if your block was validated.
